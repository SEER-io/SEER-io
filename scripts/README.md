# SEER Scripts

A collection of tools for network management, deployment, and automation.

## Key Scripts

### 1. `setup-ci.sh`
The primary onboarding tool. Automates the connection between a forked GitHub repo and a Cloudflare account. It handles secret encryption and credential validation.

### 2. `local_node_runner.js` (Coming Soon)
A local runtime for SEER Bot Nodes. Allows operators to run nodes on their own machines (PC/Android Termux) for higher performance or testing.

### 3. `update_secrets.py`
A secure helper for managing GitHub repository secrets programmatically.
